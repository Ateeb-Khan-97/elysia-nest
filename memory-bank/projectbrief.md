# Project brief

## Project
**elysia-nest** – NestJS-like API on [Elysia](https://elysiajs.com) as the HTTP adapter.

## Summary
- Write with decorators (`@Module`, `@Controller`, `@Get`, `@Injectable`, `@Body`, `@Param`, etc.).
- Run on Elysia’s stack (Bun). No Express or NestJS runtime.

## Tech stack
- **Runtime:** Bun
- **HTTP framework:** Elysia
- **Language:** TypeScript (experimentalDecorators, emitDecoratorMetadata)
- **Notable deps:** Prisma, jose, zod, @elysiajs/openapi

## Key commands (Bun)
- `bun install`
- `bun run dev` – watch mode
- `bun run example` / `bun test`
